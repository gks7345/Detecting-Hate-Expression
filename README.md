# Detecing-Hate-Expression
YOLO-v8 based Detecting Hand Gestures of Hate Expression.

Now one of important social isuses in korea is hate expression about a specific sites.

Then this project was generated custom model for hand gestures of hate expression.

Using this model can detect hate expression and display result on bounding box.

The dataset consists of train data, valid data, test data.

Valid data consists of 10% of the total data, while the train data consists of the remaining 90%.

The test data consists of 10% of the train data.
